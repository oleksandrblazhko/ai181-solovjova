### Підготувати структури даних для імітації Token-механізму аутентифікації в СУБД PostgreSQL.

`DROP TABLE IF EXISTS sso_tokens;
CREATE TABLE sso_tokens (
		user_id INTEGER,
		token 	VARCHAR,
		FOREIGN KEY (user_id) REFERENCES users(user_id)
);`
